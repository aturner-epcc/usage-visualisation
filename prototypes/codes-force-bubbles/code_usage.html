Application usage over past month

This page shows the usage broken down by application over the last month. If you have JavaScript enabled, you will be able to use an interactive visualisation to explore the data; otherwise you will just see a table listing the application usage statistics.

In the visualisation below, each bubble represents a particular application. The radius is proportional the the total amount of time used by that application over the past month and the colour represents the number of unique users of that application. If you hover over a particular bubble, you will see the application name and the numerical values associaed with that application's usage. Clicking the buttons across to the top allows you to categorise the applications by different criteria.
